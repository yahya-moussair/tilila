# Tilila IA reference (Dual-doc source of truth)

This project uses **two official information-architecture documents** as reference and must stay consistent with both:

- `Arborescence site web Tilila.docx` (original arborescence)
- `Tilila_Arborescence_v2.docx` (v2 — May 2026, reference UX/IA)

When the documents conflict, we follow this rule:

- **Keep the v2 home structure (Zones A–D)**
- **Keep any additional modules explicitly required by the original arborescence** as long as they do not break v2 rules (example: add modules after Zone D; avoid adding CTAs inside Zone A unless explicitly required)

---

## Global navigation (top-level)

- **Home**: `/`
- **About**: `/about`
- **Events**: `/events`
- **Experts**: `/experts`
- **Learn**: `/learn`
- **Media (CPD)**: `/media` (reachable via Experts pillar and internal links)

Primary CTA in navigation:
- **Find an expert** → `/experts`

## Home (ACCUEIL)

### v2 Zones (must exist)

- **Zone A — Hero**: featured video + institutional message
  - v2 says: **no CTA inside zone A**
- **Zone B — Actualités**: max 3 CMS-driven cards (typed)
  - Fields: title (tri-lang), type, date, link
  - Rule: **max 3 active**
- **Zone C — Stats + CTA**: 3 metrics + primary CTA “Find an expert”
- **Zone D — 4 pillar tiles**: About / Events / Experts / Learn

### Original arborescence modules (must exist)

These can be added **after Zone D** to preserve v2:

- **Featured experts** (“Expertes à la Une”)
- **Latest media** (“Derniers contenus médias”)
- **Quick agenda** (“Agenda rapide”)
- **Partners** (“Partenaires”)

Footer must include:
- Social networks, Newsletter, Contact, Plan du site, Mentions légales, Copyright SOREAD 2M

## About (ABOUT)

Required sections (combined):

- Tilila: Vision / Mission / Positionnement / Valeurs EDI (Parité / Diversité / Inclusion)
- SOREAD 2M: Présentation groupe / Média citoyen / Historique engagements / CPD
- Programme EDI Tilila: Objectifs 2026 / Axes / KPIs / Actions internes & externes
- Gouvernance: Présidence / Bureau CPD / Membres permanents / Partenaires / Contacts
- Presse & contact: Dossier presse / Logos / Contact média / Formulaire
- Références & légal: Études / Chartes EDI-ESG-RSE / Baromètre / Kit média / RGPD-CNDP / Mentions légales

## Events (TILILA EVENTS)

- Annual overview + calendar
- Tilila Awards: Concept / Prix / Critères / Jury / Candidature / FAQ / Sponsors / Historique + Lauréats par année
- Tililab: same structure as Awards (common IA)
- Tilitalks: Concept / Agenda / Speakers / Replays / Photos / Inscription

v2 simplification note:
- Agenda is kept mainly under Learn, but Events still provides calendar/overview view.

## Experts (TILILA EXPERTES)

- Directory: Morocco / Africa / Diaspora
- Search:
  - simple
  - advanced: name / domain / city / country / language
- “Experts to feature”
- Tilila Connect: journalist matchmaking / interview request / speaker panel / member network
- Contribute: join base / update profile / propose tribune
- Expert content: tribunes / interviews / analyses / opinions
- CPD Media: articles / podcasts / newsletters / video capsules / playlists
- Debates: themes / replays / guests / archives
